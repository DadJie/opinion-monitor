## 🚀 部署流程

### 1. 本地测试
```bash
cd opinion-monitor
python3 -m http.server 8000
# 访问 http://localhost:8000
```

### 2. GitHub 部署
```bash
./deploy.sh
# 按提示完成配置
```

### 3. 数据更新
```bash
# 自动：每 30 分钟（GitHub Actions）
# 手动：Actions → Update Hot News → Run workflow
```
