# 详情页事件内容展示 - 更新说明

## 📋 更新内容

在事件详情页面新增了**事件详情**展示模块，位于页面顶部，在事件概要之前。

## ✨ 功能特性

### 1. 智能内容加载
- **优先显示**：如果数据中有 `content` 字段，直接显示完整内容
- **自动生成**：如果没有 `content` 字段，自动生成详细内容

### 2. 自动生成内容结构

自动生成的内容包括：

#### 📍 事件概述
- 事件标题
- 分类和来源
- 发布时间
- 热度指数
- 情感倾向

#### 🏷️ 关键词
- 展示事件的关键词标签
- 美观的标签样式

#### 📊 舆情态势
- 传播趋势（上升/下降/平稳）
- 风险等级评估
- 应对建议

#### 💡 关注建议
- 持续关注官方通报
- 注意媒体倾向和网民情绪
- 警惕谣言扩散
- 做好应对准备

### 3. 加载状态提示
- 显示"正在加载事件详情..."动画
- 加载完成后自动隐藏

## 🔧 技术实现

### 数据获取
```javascript
// 优先从 API 获取完整数据
const response = await fetch(`${API_BASE_URL}/api/opinions/${opinionId}`);
const opinion = await response.json();
```

### 内容渲染
```javascript
// 有 content 字段
if (opinion.content) {
  contentEl.innerHTML = opinion.content;
} else {
  // 自动生成详细内容
  contentEl.innerHTML = generateDetailedContent(opinion);
}
```

### 降级模式
- API 失败时自动切换到本地 `data/opinions.json`
- 确保页面始终可访问

## 🚀 使用方法

### 1. 启动后端 API
```bash
cd backend
python3 main.py
```

### 2. 访问详情页
在首页点击任意舆情卡片，或直接在浏览器访问：
```
http://localhost:8001/detail.html?id=1
```

### 3. 查看内容
- 页面加载后会自动获取事件详情
- 内容显示在"事件详情"模块中
