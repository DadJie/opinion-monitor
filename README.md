# 舆情眼 (PublicOpinion Eye)

国内重点舆情实时监控平台 - 可视化展示热点舆情、事件分类、关键节点和分析判断。

## 🎯 项目简介

**舆情眼** 是一个专注于国内舆情监控的可视化平台，提供：

- 📊 **实时热点** - 聚合全网热点舆情，按热度排序
- 🏷️ **智能分类** - 社会、经济、科技、娱乐等多维度分类
- ⏱️ **事件时间线** - 展示事件发展的关键节点
- 🤖 **AI 分析** - 情感倾向、风险评估、趋势预测

## 🚀 快速开始

### 方法一：本地运行（演示）

```bash
# 进入项目目录
cd opinion-monitor

# 使用 Python 简易服务器
python3 -m http.server 8000

# 访问 http://localhost:8000
```

### 方法二：部署到 GitHub（推荐，含实时数据）

```bash
# 一键部署脚本
chmod +x deploy.sh
./deploy.sh
```

然后按照提示操作，5 分钟完成部署！

详细步骤请查看 [DEPLOY.md](DEPLOY.md)

## 📝 使用说明

### 查看舆情详情
1. 在首页点击任意舆情卡片
2. 自动跳转到详情页查看完整信息
3. 查看时间线和 AI 分析

### 筛选舆情
1. 点击分类按钮筛选特定类别
2. 使用搜索框输入关键词
3. 点击热词云图中的词汇快速搜索

### 查看数据看板
1. 点击导航栏"数据看板"
2. 查看情感分布、分类统计等可视化图表
