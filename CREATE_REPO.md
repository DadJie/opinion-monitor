# ⚠️ 需要先在 GitHub 创建仓库

## 📋 快速步骤（1 分钟完成）

### 步骤 1：创建空仓库

1. **访问**：https://github.com/new

2. **填写信息**：
   - Owner: `DadJie`
   - Repository name: `opinion-monitor`
   - Description: `国内重点舆情实时监控平台`
   - ✅ Public（公开）
   - ❌ **不要**勾选 "Add a README file"
   - ❌ **不要**勾选 ".gitignore"
   - ❌ **不要**选择 License

3. **点击**：**Create repository**

---

### 步骤 2：告诉我已完成

创建完成后，回复我 **"已创建"**，我会立即帮你推送代码。

---

### 步骤 3：推送完成后的配置

推送成功后，你需要：

1. **启用 Actions**
   - 访问：https://github.com/DadJie/opinion-monitor/actions
   - 点击 "I understand my workflows, go ahead and enable them"

2. **配置 Pages**
   - Settings → Pages
   - Source: GitHub Actions
   - Save

3. **触发数据更新**
   - Actions → Update Hot News → Run workflow

4. **部署网站**
   - Actions → Deploy to GitHub Pages → Run workflow

5. **访问网站**
   - https://DadJie.github.io/opinion-monitor/
