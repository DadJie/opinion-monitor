# 舆情监控系统 - 执行计划
创建时间：2026-03-19 22:43

## 项目名称
**舆情眼 (PublicOpinion Eye)** - 国内重点舆情实时监控平台

## 目标
构建一个实时展示国内热点舆情、事件分类、关键节点和分析判断的 Web 平台。

## 核心功能

### 1️⃣ 首页 - 舆情总览
- 实时热点舆情列表（按热度排序）
- 舆情分类筛选（政治、经济、社会、科技、娱乐等）
- 热度趋势图表
- 24 小时舆情播报

### 2️⃣ 舆情详情页
- 事件概述
- 时间线/关键节点展示
- 传播路径可视化
- 情感分析（正面/负面/中性）
- AI 分析判断
- 相关舆情关联

### 3️⃣ 数据看板
- 热词云图
- 地域分布图
- 平台来源统计
- 情感倾向占比

## 技术栈

| 层级 | 技术选型 |
|------|---------|
| **前端** | HTML5 + CSS3 + JavaScript (原生) |
| **图表** | Chart.js / ECharts |
| **后端** | Python (FastAPI) |
| **数据源** | TrendRadar2 + 自定义爬虫 |
| **数据库** | SQLite (轻量) / PostgreSQL (生产) |
| **部署** | Docker + Nginx |

## 设计风格

- **配色**: 深蓝 (#1a365d)、中国红 (#c53030)、科技蓝 (#3182ce)、中性灰 (#4a5568)
- **字体**: 系统字体栈
- **布局**: 响应式设计，支持桌面和移动端
- **交互**: 卡片式布局，悬停效果，平滑过渡

## 数据源

1. **微博热搜** - 社会热点
2. **知乎热榜** - 深度讨论
3. **抖音热榜** - 短视频舆情
4. **百度热搜** - 搜索趋势
5. **今日头条** - 新闻资讯
6. **澎湃新闻** - 权威媒体
7. **财联社** - 财经舆情

## AI 分析维度

1. **情感倾向** - 正面/负面/中性
2. **热度预测** - 上升趋势/下降趋势/平稳
3. **风险评估** - 高/中/低风险等级
4. **关键节点** - 事件转折点识别
5. **关联分析** - 相关事件推荐
